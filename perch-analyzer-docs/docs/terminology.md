# Terminology

- ARU: autonomous recording unit
- Recording: a recording from an ARU
- Window: a section of audio from a recording, usually 3-5 seconds in length
- Possible example: a window temporarily labeled as a particular species as a result of a search
- Annotation: a label (such as a bird species) placed on a window
- [Spectrogram](https://en.wikipedia.org/wiki/Spectrogram): a visual representation of a window
